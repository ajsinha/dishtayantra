The Machinery

Everything between the tick and the ledger.

Each subsystem is configuration-switched, fail-fast, and observable — change a property, not your code.

Compute

Polyglot calculators

Run nodes in Python, C++ (pybind11), Rust (PyO3), or Java (Py4J) inside one graph — with LMDB zero-copy exchange for payloads where serialization is the bottleneck.

Transport

20+ broker integrations

Kafka, RabbitMQ, ActiveMQ, TIBCO EMS, IBM MQ, Redis, gRPC, REST, SQL — plus native object-store pub/sub on s3://, azureblob://, and gcs://.

Resilience

Connectors that ride out outages

One flag — "resilient": true — adds exponential-backoff retry, automatic reconnection, and FIFO outage buffering, so a broker blip never reorders your fills.

Continuity

High availability, your way

Primary election over Zookeeper, Redis leases, S3 leases, or a bare exclusive socket. The new primary resumes exactly the DAGs the old one suspended.

Storage

Graphs live where you do

DAG definitions on the local filesystem, Amazon S3, Azure Blob, or Google Cloud Storage — switched by a single property, shared across every instance.

Operations

Watch it work

Visual DAG designer, Prometheus metrics, health probes, live log streaming, a process-parallel worker pool, and database-backed users with hashed credentials and API keys.

Market-Aware Scheduling

It knows when the bell rings —
and when it doesn't.

Time windows that wrap midnight, weekday rules, and holiday calendars composed per DAG. Built because trading systems shouldn't compute into a closed market.

config/dags/us_equities_session.json
{
  "name": "us_equities_session",
  "start_time": "0930",
  "duration": "6h30m",
  "schedule": {
    "days_of_week": ["mon","tue","wed","thu","fri"],
    "holiday_calendars": ["USA", "CANADA"]
  }
  // Edit this file intraday — the running
  // system honors it within five minutes.
}
  • Follow several calendars at once. A cross-listed strategy following USA and CANADA stands down whenever either market closes.
  • Fifteen years pre-computed. NYSE and TSX holiday sets for 2026–2040 ship as plain JSON — extend or regenerate them with one script.
  • Overnight windows that actually work. 22:00–06:00 wraps midnight correctly, normalized to minutes-since-midnight.
  • Live without restarts. Calendar files and DAG schedules reload automatically; a broken edit is rejected and the last good rules stay in force.
