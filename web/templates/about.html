What is DishtaYantra?

DishtaYantra (Sanskrit: दिष्टयन्त्र - "Directed Instrument") is a powerful, enterprise-grade Directed Acyclic Graph (DAG) execution engine designed for real-time data processing and workflow orchestration.

Built with Python and Flask, DishtaYantra provides a flexible framework for creating complex data pipelines that can:

  • Ingest data from multiple sources including Kafka, RabbitMQ, ActiveMQ, TIBCO EMS, WebSphere MQ, REST APIs, Files, and more
  • Process and transform data using customizable calculators and transformers
  • Publish results to various destinations in real-time
  • Scale horizontally using the AutoClone feature for high-throughput scenarios
  • Schedule execution using time windows for controlled processing periods

Key Features

Visual DAG Designer

Create and edit DAG configurations visually with our intuitive drag-and-drop designer. No coding required for basic workflows.

Multi-Protocol Support

Connect to Kafka, RabbitMQ, ActiveMQ, TIBCO EMS, IBM MQ, Redis, REST APIs, SQL databases, gRPC, and more.

Time Windows

Schedule DAG execution within specific time windows. Perfect for batch processing during off-peak hours or market hours.

AutoClone Scaling

Automatically scale DAG instances up or down based on time schedules. Handle peak loads without manual intervention.

In-Memory Cache

Redis-compatible in-memory cache for fast data access. Share state between DAG nodes with persistent snapshots.

Role-Based Access

Multi-role user system with Admin, Operator, and User roles. Control who can view, operate, or manage DAGs.
