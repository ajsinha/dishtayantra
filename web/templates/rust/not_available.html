Rust Integration Not Available

The Rust calculator module is not currently loaded or available.

Possible Reasons:
  • The dishtayantra_rust.so module hasn't been built yet
  • PyO3 is not installed
  • The module file is not in Python's path
  • Rust Manager is disabled in config/rust_config.json
How to Build:
# Install maturin (build tool for PyO3)
pip install maturin

# Build the Rust module
cd rust
maturin develop --release
Prerequisites:
  • Rust 1.70 or later: curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • maturin: pip install maturin
  • PyO3 (added to Cargo.toml)
Configuration:

Ensure config/rust_config.json exists and contains:

{
  "rust_manager": {
    "enabled": true,
    "auto_load_on_startup": true
  }
}
