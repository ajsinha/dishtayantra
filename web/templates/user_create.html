User Management Guidelines
Username Best Practices
  • Use lowercase letters
  • Keep it simple and memorable
  • Avoid special characters except dots, underscores, hyphens
  • Use format: firstname.lastname or role-based names
Password Security
  • Minimum 6 characters (longer is better)
  • Mix uppercase and lowercase letters
  • Include numbers and special characters
  • Avoid common words or patterns
Role Assignment
  • User: For read-only access
  • Operator: For operational tasks
  • Admin: For system administration
  • Multiple roles can be combined
Security Tips
  • Review user roles regularly
  • Remove unused accounts promptly
  • Use principle of least privilege
  • Monitor admin account activity
