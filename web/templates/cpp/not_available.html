C++ Integration Not Available

The C++ calculator module is not currently loaded or available.

Possible Reasons:
  • The dishtayantra_cpp.so module hasn't been built yet
  • pybind11 is not installed
  • The module file is not in Python's path
  • CPP Manager is disabled in config/cpp_config.json
How to Build:
cd cpp
mkdir build && cd build
cmake ..
make
Prerequisites:
  • CMake 3.14 or later
  • C++17 compatible compiler (GCC 7+, Clang 5+, MSVC 2017+)
  • pybind11: pip install pybind11
  • Python development headers
Configuration:

Ensure config/cpp_config.json exists and contains:

{
  "cpp_manager": {
    "enabled": true,
    "auto_load_on_startup": true
  }
}
